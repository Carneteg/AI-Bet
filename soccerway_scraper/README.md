# Soccerway Data Crawler for AI Pipelines

A robust, production-quality Python crawler designed to ingest quantitative match data from Soccerway.com. Built specifically to generate raw training and validation datasets for downstream AI analysis.

## Key Features
- **Strictly Manual Execution**: No background jobs, no cron, runs precisely when invoked via CLI.
- **Robust Network Error Handling**: Internal exponential backoff and randomized retries to bypass connection drops.
- **Bot Mitigation**: Rotates dynamic User-Agents and explicitly obeys `robots.txt` rules using `urllib`.
- **Fault-Tolerant DOM Parsing**: If Soccerway changes its HTML tables or hides a metric, the scraper will yield `None` instead of hard crashing.
- **Polite Delays**: Randomizes the sleep gap between requests protecting against blanket IP bans.

## Setup Instructions

1. Ensure Python 3.9+ is installed on your system.
2. Open terminal in this folder (`soccerway_scraper`).
3. (Optional but recommended) Create a virtual environment:
   ```bash
   python -m venv venv
   # Windows: venv\Scripts\activate
   # Mac/Linux: source venv/bin/activate
   ```
4. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## How to Run Manually

The crawler offers a clean CLI powered by `argparse`.

### Basic Run
Pick a seed URL (like a specific league's competition page or a direct match page) and launch:

```bash
python main.py --seed-url https://int.soccerway.com/matches/2024/05/19/england/premier-league/arsenal-fc/everton-fc/4081559/ --max-matches 1
```

### Advanced Config Run
Customize rate-limiting and output paths:

```bash
python main.py --seed-url https://int.soccerway.com/national/england/premier-league/20232024/regular-season/r76443/ --max-matches 50 --delay-min 2.5 --delay-max 6.0 --output-file data/premier_league_raw.json
```

## AI Pipeline Usage
The output generated by this scraper is normalized JSON. It is designed to be fed into your AI engine (`lib/analyzer.ts`) back in the Next.js stack, or sent directly into a Pandas DataFrame for modeling Poisson equations, building neural networks on xG drift, or evaluating the exact historical statistics of your current "Sharp Bettor" logic.

## Known Limitations
- Data like minute-by-minute line-movement, granular player heatmaps, or hidden injury stats are heavily nested inside dynamically rendered JavaScript blocks. 
- This scraper utilizes purely `requests` + `BeautifulSoup` to target the raw HTML Box Score payload. If heavily gated JS content is required in the future, the `core/fetcher.py` will need an upgrade to `Playwright`.
